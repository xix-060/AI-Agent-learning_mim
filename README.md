# AI-Agent-learning\_mim

## AI Agent 学习仓库

> 个人 AI Agent 入行学习记录，按 12 周计划推进。

## 🚀 快速开始

```bash

# 1. 克隆仓库
git clone https://github.com/xix-060/AI-Agent-learning_mim.git
cd ai-agent-learning_mim

# 2. 配置环境
bash scripts/env.sh
conda activate ai-agent

# 3. 配置 API Key
cp .env.example .env
# 编辑 .env 填入你的 API Key

# 4. 运行
python hello_llm.py

'''bash
