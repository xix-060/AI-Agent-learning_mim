## 三、矩阵维度标注

### 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| n | 输入序列长度 | 句子有多少个词 |
| d_model | 512 | 模型总维度 |
| h | 8 | 注意力头数 |
| d_k | 64 = d_model / h | 每个头的 Q/K 维度 |
| d_v | 64 = d_model / h | 每个头的 V 维度 |

### 各矩阵形状

手写时画成这样：

```
输入 X :              (n, 512)     ← 一句话，每个词512维

权重矩阵:
  W^Q :              (512, 64)     ← 把512维压缩到64维
  W^K :              (512, 64)
  W^V :              (512, 64)
  W^O :              (512, 512)    ← 8头拼接后，再还原回512维

投影后:
  Q :                (n, 64)       ← X · W^Q
  K :                (n, 64)       ← X · W^K
  V :                (n, 64)       ← X · W^V

计算过程:
  QK^T :             (n, n)        ← ★ 这就是 attention 矩阵!
  QK^T / sqrt(d_k) : (n, n)       ← 缩放后还是 n×n
  softmax(...) :     (n, n)        ← 每行变成概率分布(和为1)
  softmax(...) · V : (n, 64)       ← 最终单头输出

多头拼接:
  Concat(head_1...head_8) : (n, 512)   ← 8 × 64 = 512
  · W^O :                     (n, 512) ← 最终输出，和输入同维度
```

    │─────────────→│  W^K  │─────→│   K   │──┘  │  attention  │

    │              ┌───────┐      ┌───────┐           │
    │─────────────→│  W^V  │─────→│   V   │           │
    │              │(512,64)│     │(n,64) │           │
